# badibanga.top — Portfolio de James BADIBANGA (Criho)

Portfolio personnel de **James BADIBANGA (Criho)** — Développeur Web Fullstack & Consultant en Intégration d'IA, basé à Kinshasa (RD Congo).

Site 100% statique (HTML/CSS/JS vanilla), sans étape de build, déployé sur Vercel.

## Structure

```
index.html              # page unique (À propos, Parcours, Portfolio, Contact)
assets/css/style.css    # thème vCard + sections personnalisées (stats, tags, CTA)
assets/js/script.js     # navigation par onglets, filtres portfolio, validation formulaire
assets/images/          # photo, favicon, vignettes projets (projects/)
vercel.json             # clean URLs + en-têtes de sécurité
docs/                   # documentation technique
```

## Développement local

Aucune dépendance à installer. Ouvrir `index.html` dans un navigateur, ou servir le dossier :

```bash
npx serve .
```

## Déploiement

Le site est déployé sur Vercel ; chaque push sur `main` déclenche un déploiement.
